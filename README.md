# Store Web Project
 
A demo project of an e-commerce website including the basics of ASP.NET Core MVC.

Project Content
--------------------------------------------------------------------------------------

◆ MVC

◆ Code First Approach

◆ Services

◆ Admin Panel

◆ Filtering and Searching

◆ Bootstrap

◆ Session Management

◆ Custom Tag Helpers

◆ Identity

◆ File Operations

◆ AutoMapper
